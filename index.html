Welcome to Pandaemic
Covid-19/ SARS II has affected the world substantially and the new hot thing is social distancing.
But how do these measures affect infection rate (and subsequently the outbreaks outcome)?
To better understand this, I created Pandaemic, a pandemic simulator that allows you to play with a lot of parameters to build a more intuitive understanding of the underlying mechanics.
See the github page for more information.
